app/exceptions.py
----------------------------------------------------------------------------
Расположение: app/exceptions.py
Смысловая нагрузка: Определение кастомных HTTP-исключений для приложения.
                 Архитектурно - централизованное хранилище ошибок API.
Внутренние импорты: fastapi.status, fastapi.HTTPException
Процедуры:
- UserAlreadyExistsException: Пользователь уже существует (409)
- ServerAlreadyExistsException: Сервер уже существует (409)
- ServerUrlAlreadyExistsException: URL сервера уже существует (409)
- UserNotFoundException: Пользователь не найден (404)
- UserIdNotFoundException: Отсутствует ID пользователя (404)
- IncorrectEmailOrPasswordException: Неверный логин/пароль (400)
- TokenExpiredException: Токен истек (401)
- InvalidTokenFormatException: Некорректный формат токена (400)
- TokenNoFound: Токен отсутствует в заголовке (401)
- NoJwtException: Невалидный JWT токен (401)
- NoUserIdException: Не найден ID пользователя (404)
- ForbiddenException: Недостаточно прав (403)
- ServerNotFoundException: Сервер не найден (404)
- UserServerNotFoundException: Сервер не добавлен пользователю (404)

АРХИТЕКТУРА ПРОЕКТА
=============================================================================

Многоуровневая архитектура:
1. ROUTER (app/auth/router.py, app/game/router.py, app/views/router.py)
   - API endpoints и представления

2. SCHEMAS (app/auth/schemas.py, app/game/schemas.py)
   - Валидация данных (Pydantic)

3. DAO (app/auth/dao.py, app/game/dao.py, app/dao/base.py)
   - Доступ к данным

4. MODELS (app/auth/models.py, app/game/models.py)
   - SQLAlchemy модели (слой БД)
