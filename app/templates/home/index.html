О проекте

Это стартовая сборка FastAPI с интегрированной SQLAlchemy 2. Проект включает аутентификацию, авторизацию и управление пользователями.

Быстрый старт

Используйте API эндпоинты для регистрации и входа:

  • POST /auth/register/ — регистрация пользователя
  • POST /auth/login/ — вход в систему
  • GET /auth/me/ — получить данные текущего пользователя
